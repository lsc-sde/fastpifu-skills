# Adult trigger digit

<!-- concept_ids: (completed by data team) -->

## Scope

Day-case surgery for adult trigger digit after up to 2 cortisone injections.

## PIFU — suitable for patient-initiated follow-up

- PIFU 3 months only if the patient is not progressing as expected.

## RETAIN_FOLLOW_UP — timed specialist follow-up still needed

- Wound therapy/review only if patient/surgical team are concerned; suture removal at GP/local arrangements if required.
- Not progressing as expected.

## DISCHARGE — no ongoing specialist follow-up needed

- Progressing as expected — discharge.

## Surveillance / review interval

PIFU 3 months (conditional).
